ID: SOL-CALC2-042 Topic: Integration by Parts
Reviewer Status VERIFIED BY FACULTY [OK]
Original Question
Evaluate the following indefinite integral using the method of integration by parts:
∫ x² ln(x) dx

Step-by-Step Analysis

Official Solution
01.

Choose variables u and dv using the LIATE rule.

u = ln(x) dv = x² dx
du = (1/x) dx v = x³/3

warning Common Trap: Don't flip u and dv! Logarithms (L) always take priority for 'u' over Algebraic (A) terms.

02.

Apply the Integration by Parts formula: ∫ u dv = uv - ∫ v du

∫ x² ln(x) dx = (ln x)(x³/3) - ∫ (x³/3)(1/x) dx
= (x³/3) ln x - (1/3) ∫ x² dx
03.

Final integration and constant of integration.

= (x³/3) ln x - (x³/9) + C

Don't forget the +C! Required for all indefinite integrals.
