## 3. DETAILED SECTION ANALYSIS WITH LEARNING OBJECTIVES

### Chapter 3: Techniques of Integration (Priority Chapter)

#### **3.1 Integration by Parts**
**Learning Objectives:**
1. Recognize when to use integration by parts
2. Use the integration-by-parts formula to solve integration problems
3. Use the integration-by-parts formula for definite integrals

**Key Topics:**
- Integration-by-parts formula: ∫u dv = uv - ∫v du
- LIATE rule for choosing u (Logarithmic, Inverse trig, Algebraic, Trigonometric, Exponential)
- Repeated integration by parts
- Reduction formulas

**Problem Types:**
- Polynomial × exponential functions
- Polynomial × trigonometric functions
- Logarithmic integrals
- Inverse trigonometric integrals
- Cyclic integration by parts (exponential × trigonometric)

**Exercise Count:** ~62 problems (Section 3.1 Exercises)

---

#### **3.2 Trigonometric Integrals**
**Learning Objectives:**
1. Solve integration problems involving products and powers of sin(x) and cos(x)
2. Solve integration problems involving products and powers of tan(x) and sec(x)
3. Use reduction formulas to solve trigonometric integrals

**Key Topics:**
- Integrating products of sines and cosines
- Power-reduction formulas
- Products of tangents and secants
- Trigonometric identities for integration
- Double-angle formulas

**Problem Types:**
- ∫sin^m(x)cos^n(x)dx (various combinations of odd/even powers)
- ∫tan^m(x)sec^n(x)dx
- Using trigonometric identities to simplify
- Reduction formulas for powers

**Exercise Count:** ~56 problems

---

#### **3.3 Trigonometric Substitution**
**Learning Objectives:**
1. Solve integration problems involving the square root of a sum or difference of two squares

**Key Topics:**
- Substitution for √(a² - x²) → x = a sin(θ)
- Substitution for √(a² + x²) → x = a tan(θ)
- Substitution for √(x² - a²) → x = a sec(θ)
- Reference triangles for back-substitution
- Definite integrals with trig substitution

**Problem Types:**
- Integrals involving √(a² - x²), √(a² + x²), √(x² - a²)
- Combinations with powers of x
- Definite integrals requiring trigonometric substitution
- Arc length problems

**Exercise Count:** ~40 problems

---

#### **3.4 Partial Fractions**
**Learning Objectives:**
1. Integrate a rational function using the method of partial fractions
2. Recognize simple linear factors in a rational function
3. Recognize repeated linear factors in a rational function
4. Recognize quadratic factors in a rational function

**Key Topics:**
- Partial fraction decomposition
- Distinct linear factors
- Repeated linear factors
- Irreducible quadratic factors
- Long division for improper fractions

**Problem Types:**
- Decomposing rational functions
- Integrating with distinct linear factors
- Integrating with repeated linear factors
- Integrating with irreducible quadratics
- Mixed partial fraction decompositions

**Exercise Count:** ~48 problems

---

#### **3.5 Other Strategies for Integration**
**Learning Objectives:**
1. Use a table of integrals to solve integration problems
2. Use computer algebra systems (CAS) to solve integration problems

**Key Topics:**
- Using integral tables
- Strategies for integration (general approach)
- CAS integration tools

**Problem Types:**
- Reference table usage
- Strategy selection problems

---

#### **3.6 Numerical Integration**
**Learning Objectives:**
1. Approximate the value of a definite integral by using the midpoint and trapezoidal rules
2. Determine the absolute and relative error in using a numerical integration technique
3. Estimate the absolute and relative error using an error-bound formula
4. Recognize when the midpoint and trapezoidal rules over- or underestimate the true value

**Key Topics:**
- Midpoint rule
- Trapezoidal rule
- Simpson's rule
- Error bounds for numerical integration
- Comparing accuracy of methods

**Problem Types:**
- Approximating definite integrals numerically
- Error estimation
- Comparing methods
- Determining required n for desired accuracy

---

#### **3.7 Improper Integrals** ⭐ HIGH FREQUENCY
**Learning Objectives:**
1. Evaluate an integral over an infinite interval
2. Evaluate an integral over a closed interval with an infinite discontinuity within the interval
3. Use the comparison theorem to determine convergence of improper integrals

**Key Topics:**
- Type I: Infinite intervals (∫[a,∞), ∫(-∞,b], ∫(-∞,∞))
- Type II: Discontinuous integrands
- Comparison theorem for convergence
- Absolute convergence
- p-integrals and their convergence properties

**Problem Types:**
- Convergence/divergence determination
- Evaluating convergent improper integrals
- Using comparison theorem
- Combination of types
- Limits with infinite discontinuities

**Exercise Count:** ~60 problems

### Chapter 4: Differential Equations (Priority Chapter)

#### **4.1 Basics of Differential Equations**
**Learning Objectives:**
1. Identify the order of a differential equation
2. Explain what is meant by a solution to a differential equation
3. Distinguish between the general solution and a particular solution
4. Identify an initial-value problem

**Key Topics:**
- Order and degree of differential equations
- General vs. particular solutions
- Initial-value problems (IVP)
- Verifying solutions

---

#### **4.2 Direction Fields and Numerical Methods** ⭐ HIGH FREQUENCY
**Learning Objectives:**
1. Draw the direction field for a given first-order differential equation
2. Use a direction field to draw a solution curve of a first-order differential equation
3. Use Euler's method to approximate solutions

**Key Topics:**
- Direction fields (slope fields)
- Euler's method for numerical approximation
- Visualizing solutions

---

#### **4.3 Separable Equations**
**Learning Objectives:**
1. Use separation of variables to solve a differential equation
2. Solve applications using separation of variables

**Key Topics:**
- Separation of variables technique
- Solving for general solutions
- Applying initial conditions
- Applications: population growth, mixing problems

---

#### **4.4 The Logistic Equation**
**Learning Objectives:**
1. Describe the concept of environmental carrying capacity in the logistic model
2. Draw a direction field for a logistic equation
3. Solve the logistic differential equation

**Key Topics:**
- Logistic growth model
- Carrying capacity
- Solving logistic equations
- Applications to population dynamics

---

#### **4.5 First-order Linear Equations** ⭐ HIGH FREQUENCY
**Learning Objectives:**
1. Write a first-order linear differential equation in standard form
2. Find an integrating factor and use it to solve a first-order linear differential equation

**Key Topics:**
- Standard form: dy/dx + P(x)y = Q(x)
- Integrating factor method
- Applications to mixing problems, cooling, circuits
